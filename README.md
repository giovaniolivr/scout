# Scout

O **Scout** é uma plataforma desenvolvida com **Django**, voltada para conectar **jovens talentos e empresas** de forma justa e colaborativa, através de um sistema de **feedbacks e pontuações** que valoriza tanto **soft skills** quanto **hard skills**.

## Tecnologias utilizadas
- Python 3.10+
- Django 5+
- HTML5, CSS3 e Bootstrap
- JavaScript

## Como rodar o projeto localmente
```bash
# Clonar o repositório
git clone https://github.com/giovaniolivr/scout.git

# Acessar a pasta
cd scout

# Criar e ativar o ambiente virtual
python -m venv venv
venv\Scripts\activate  # (Windows)
source venv/bin/activate  # (Linux/macOS)

# Instalar dependências
pip install -r requirements.txt

# Rodar o servidor
python manage.py runserver
```
